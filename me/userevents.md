### **me/userevents**

Deployed on branch: `api` `dev`

#### **Description**

Returns userevents for the logged in user

#### **Method**

GET

#### **Param**

- total
- offset
- limit
- min_year
- max_year
- data

Example:
http://api.charged.fm/me/userevents

```javascript

{
    "total": 0,
    "offset": 0,
    "limit": 1000,
    "min_year": 0,
    "max_year": 0,
    "data": []
}
```
