### **me/followingperformers**

Deployed on branch: `api` `dev`

#### **Description**

Returns all performers that the logged-in user is following

#### **Method**

GET

#### **Param**

- total
- offset
- limit
- data

Example:
http://api.charged.fm/me/followingperformers

```javascript

{
total: 0,
offset: 0,
limit: 10,
data: [ ]
}

```
