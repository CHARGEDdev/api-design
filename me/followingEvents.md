### **me/followingevents**

Deployed on branch: `api` `dev`

#### **Description**

Returns all events that the logged-in user is following

#### **Method**

GET

#### **Param**

- total
- offset
- limit

Example:
http://api.charged.fm/me/followingevents

```javascript
{
total: 0,
offset: 0,
limit: 10,
data: [ ]
}
```
